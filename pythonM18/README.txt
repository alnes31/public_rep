
Телеграм-бот для конвертации валют.
Имя в телеграме @cryptones

Комманды бота:
1. /start стартовое сообщение с основным описанием
2. /help более подробное описание и перечисление комманд;
3. /values список известных боту валют;
4. /exchange_rates список всех курсов валют, которые известны боту
5. <валюта 1> <валюта 2> <количество> выводит значение количества валюты 2 от валюты 1.
